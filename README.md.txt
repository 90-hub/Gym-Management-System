# Gym Management System

## Project Description

This project is a Gym Management System developed in C++ using Data Structures (Linked List) and Sorting Algorithms. The system manages gym members including their personal details like name, age, and weight. It also provides sorting functionality based on weight using different algorithms.

## Features

* Add new gym members
* Display all members
* Store data using Linked List
* Dynamic memory allocation
* Sort members by weight using:

  * Bubble Sort
  * Selection Sort
  * Insertion Sort
* Menu-driven console interface

## Data Structure Used

* Singly Linked List

Each member is stored in a node:

* Name
* Age
* Weight
* Pointer to next node

## Algorithms Used

* Bubble Sort (based on weight)
* Selection Sort (based on weight)
* Insertion Sort (based on weight)

## Concepts Used

* Structures (struct Node)
* Pointers
* Dynamic Memory Allocation (new keyword)
* Linked List Operations (Insert, Traverse)
* Sorting Algorithms
* Functions (Modular Programming)
* Menu-driven Program
* Loops (for, while, do-while)
* Conditional Statements (if, switch)

## Functionalities

### Member Management

* Add gym member
* Display all members

### Sorting Features

* Sort members by weight using Bubble Sort
* Sort members by weight using Selection Sort
* Sort members by weight using Insertion Sort

## How It Works

1. The system stores gym members in a linked list.
2. New members are added dynamically at runtime.
3. Users can view all stored members.
4. Sorting options arrange members based on weight.
5. Menu allows continuous interaction until exit.

## Conclusion

This Gym Management System demonstrates the practical implementation of a Linked List and Sorting Algorithms in C++. It helps in understanding dynamic data handling, pointer manipulation, and algorithmic sorting techniques.
